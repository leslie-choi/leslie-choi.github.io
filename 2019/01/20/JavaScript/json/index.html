语法

JSON 数据的书写格式是：名称/值对。
名称/值对包括字段名称（在双引号中），后面写一个冒号，然后是值：

“name” : “吴彦祖”

JSON的取值可以是
1、数字（整数或浮点数）
2、字符串（在双引号中）
3、逻辑值（true 或 false）
4、数组（在中括号中）
5、对象（在大括号中）
6、null
